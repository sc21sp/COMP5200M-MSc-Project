# COMP5200M-MSc-Project

Football Analytics

The project aims to create and use various analytics tools to study football using a data-driven methodology. The initiative tries to quantify player performances to assess an individual based on available facts. Moreover, understand the playing patterns of a football team using intuitive graphs and animations.
Additionally, the goal is to create a scouting system dashboard to locate replacement players for certain players throughout the top five leagues by locating players with comparable player profiles. The system provides a list of players from around the world comparable to the specified player. When a player departs or is about to retire, a system like this enables teams to locate the best successor.

The objective is to create a system that suggests a number of players that matches the given player profile.
•	Build interactive data visualisations to quantify and compare individual player performances.
•	Make sense of scoring metrics such as xG to evaluate team and player performances.
•	Recreate a played match using match statistics and build a post-match analysis visual.
•	Build a player recommendation system for distinct player profiles and individual player replacements.
•	Animate specific player movements in a sequence that occurred in an actual match.

The libraries that have been used extensively throughout this project and thus necessary to be installed in the system are:
•	Matplotlib - The most well-known Python package for data visualisation. With a high degree of customisation, it enables the creation of practically any sort of chart.
•	BeautifulSoup – Beautiful Soup is a Python library for pulling data out of HTML and XML files.
•	Requests - Requests allows you to send HTTP/1.1 requests extremely easily.
•	Sklearn - A free machine learning library for Python is called Scikit-learn. It supports Python's NumPy and SciPy libraries as well as several methods, including support vector machines, random forests, and k-neighbors.
•	Scipy - A scientific computation package called SciPy is built on top of NumPy.
•	Scientific Python is a common abbreviation. It offers more helpful functions for signal processing, statistics, and optimization.
•	Numpy - NumPy is a library for the Python programming language that is used for various higher order mathematical operations over large, multi-dimensional arrays and matrices.
•	Pandas - pandas is a Python package that offers fast, flexible, and expressive data structures that are intended to make relational and labelled data, simple and intuitive.
•	Streamlit - Streamlit is a free and open-source framework to rapidly build and share machine learning and data science web apps.
•	Mplsoccer - mplsoccer is a Python library for plotting football charts in Matplotlib and loading StatsBomb open-data.
